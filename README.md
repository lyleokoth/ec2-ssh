# ec2-ssh
A GitHub ction to ssh into ec2 and start a service.

## Inputs

## `who_to_greet`

**Required** The name of the person to greet. Default `"World"`.

## Outputs

## `time`

The time we greeted you.

## Example usage

uses: actions/hello-world-docker-action@v1
with:
  who_to_greet: 'Lyle Okoth'
